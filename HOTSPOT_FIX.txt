HOTSPOT FIX - Pico WH Connection Issue
=======================================

PROBLEM DIAGNOSIS:
------------------
Dari diagnostic_network.py, ditemukan:
✅ Network interface (wlan0) - OK
❌ WiFi Hotspot (hostapd) - NOT RUNNING
✅ IP Address (10.42.0.1) - OK
✅ Firewall - OK
❌ Port 5000 - NOT LISTENING (server belum start)
❌ No connected devices
❌ Ping test failed
❌ TCP server test failed

ROOT CAUSE:
-----------
hostapd service tidak aktif, sehingga SSID "N3IWF_AQUA" tidak di-broadcast.
Pico WH tidak bisa scan dan connect ke WiFi.

SOLUTION:
---------
Ada 2 cara fix:

CARA 1: Quick Fix (Recommended)
--------------------------------
Jika hostapd sudah pernah dikonfigurasi sebelumnya:

cd ~/thesis
git pull
sudo bash fix_hotspot.sh

Script ini akan:
- Cek apakah hostapd sudah dikonfigurasi
- Jika sudah, langsung start service
- Jika belum, jalankan setup lengkap

CARA 2: Manual Fix
------------------
Jika hanya perlu start service:

sudo systemctl unmask hostapd
sudo systemctl enable hostapd
sudo systemctl start hostapd
sudo systemctl start dnsmasq

Cek status:
sudo systemctl status hostapd
sudo systemctl status dnsmasq

CARA 3: Full Setup (Jika belum pernah setup)
---------------------------------------------
cd ~/thesis
sudo bash setup_hotspot.sh

VERIFICATION:
-------------
Setelah fix, jalankan diagnostic lagi:

python3 diagnostic_network.py

Harus muncul:
✅ WiFi Hotspot - ACTIVE
✅ Port 5000 - LISTENING (setelah start server)

NEXT STEPS:
-----------
1. Fix hotspot (pilih salah satu cara di atas)
2. Verify dengan diagnostic: python3 diagnostic_network.py
3. Start server: python3 main/real/run_real.py
4. Flash firmware ke Pico WH (jika belum)
5. Test koneksi end-to-end
